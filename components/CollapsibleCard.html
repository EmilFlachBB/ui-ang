Component: CollapsibleCard

Provides an expandable details-summary/body-header view. Uses bb-collapsible-ui for a predefined template ### Accessibility ariaLabelForIcon is discernible text for chevron icon

Selector: bb-collapsible-card-ui

Inputs

Input Type Description Default
aria-label string | null Accessible label for collapsible card 'toggle'
ariaLabelForIcon string Accessible label for chevron icon. 'Chevron icon'
hasChevron boolean Whether chevron should be displayed. Defaults to true. true
highlight boolean Whether card should be highlighted. Defaults to false. false
isOpen boolean Collapsible card open state false
shadow boolean Whether card should has shadow effect. Defaults to true. true
size CollapsibleCardSize Card size. Defaults to md. 'md'

Outputs

Output Type Description
isOpenChange EventEmitter Emits when the isOpen input is changed.

Methods

toggle(): void
