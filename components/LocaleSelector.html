Component: LocaleSelector

Selector: bb-locale-selector-ui

Inputs

Input Type Description Default
btnColor string Color of the button. Defaults to primary. 'primary'
dropDownButtonLabel string Button aria label. -
dropdownPlacement PlacementArray The position of the dropdown for the locale selector, position will be picked in order of feasibility. ['bottom-end', 'bottom-start', 'top-end', 'top-start']
fullWidth boolean If true it will stretch the button inside to 100% width. false
icon string Icon that is displayed in the button. -
iconSize string The size of the icon to be displayed. Defaults to md. 'md'
showModalOnMobileView boolean If true it will show the locale selection inside a modal on mobile view. false

Outputs

Output Type Description
selectLocale EventEmitter An output property which will emit a selected Locale code.

Methods

changeLanguage(localeCode: string): void

Changes the current language to the specified locale, toggles locale modal and triggers a output event with selected locale code.

Parameters:

  • localeCode (string): undefined

toggleLocaleModal(): void

Toggles the visibility of the locale selector modal.
