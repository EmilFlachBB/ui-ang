Component: Button

Component that displays a button.

Selector: bb-button-ui

Inputs

Input Type Description Default
autofocus boolean This Boolean attribute lets you specify that the button should have input focus when the page loads, unless the user overrides it, for example by typing in a different control. Only one form-associated element in a document can have this attribute specified. false
block boolean The flag to indicate whether the button should fill the container that it is in. Defaults to 'false'. false
buttonSize "sm" | "md" Size of the button. 'md'
color ButtonColor | string The color for the button. Defaults to 'primary'. 'primary'
disabled boolean This Boolean attribute indicates that the user cannot interact with the button. Defaults to false. false
id string The id for the button. Defaults to unique string -
type ButtonType The HTML5 button type. Defaults to 'button'. 'button'
