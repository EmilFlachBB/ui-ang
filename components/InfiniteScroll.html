Component: InfiniteScroll

Component that displays an infinite scroll.

Selector: bb-infinite-scroll-ui

Inputs

Input Type Description Default
aria-label string Accessible label when control does not need to render label tag. 'Infinite scrolling content'
debounce number Specify debounce duration in ms 100
disableScrollEnd boolean If true then scrollEnd event should NOT be emitted false
focusable boolean Sets the focusability of the component. If value is true, the component becomes focusable. If value is false, the component becomes non-focusable. Component is focusable by default true
isLoading boolean Takes a boolean value to indicate the loading state of data false
offset number The number of pixels from the bottom of the element at which to trigger the scrollEnd event. 0
scrollContainer string Takes a CSS class selector to identify the scrolling element '.infinite-scroll-wrapper'

Outputs

Output Type Description
scrollEnd EventEmitter Event that is triggered when the user has scrolled to the bottom of the element.
