Component: Icon

Component that displays an icon.

Selector: bb-icon-ui

Inputs

Input Type Description Default
animate boolean Whether the icon should be animated. Defaults to false. false
aria-label string | undefined Input that can be used to specify the aria-label attribute. -
backgroundType "circle" | "square" | undefined Wether the icon has a background. If not set it will display just the icon without background. Possible values are circle and square. -
color string | undefined The color of the icon represented as a color name contained in backbase-theme. -
cropped boolean | string Whether the icon should be cropped horizontally. -
flipOnRtl boolean Whether the icon should be flipped when the page direction is RTL (right-to-left). -
inverse boolean Whether to use white background and inverse icon colors false
name string | undefined The name of the icon -
size IconSize The size of the icon represented as a size name contained in backbase-theme. This is ignored when the circle option is used. Defaults to md. 'md'
