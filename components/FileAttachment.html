Component: FileAttachment

### Accessibility Current component provide option to pass needed accessibility attributes: - id - aria-label - aria-describedby Component displays file attachment.

Selector: bb-file-attachment-ui

Inputs

Input Type Description Default
aria-describedby string Set aria-describedby with an element id that contains a detailed decription of the widget. 'file-attachment-card-body'
aria-label string Accessible label for delete button when control does not need to render label tag. 'Delete file'
block boolean The flag to indicate whether the file-attachment should fill the container that it is in. Defaults to 'false'. false
deletable boolean Show delete button flag. Defaults to false. false
disabled boolean Disabled state flag. Defaults to false. false
fileContent Observable | undefined Set file content (required if disabled property is not set to true). -
id string The id for attachment card. 'file-attachment-card-body_' + getDynamicId()
loading boolean Loading indicator flag. Defaults to false. false
name string | undefined Set file name (required). -
size number | string | undefined Set file size (required). Accepts number or string. -

Outputs

Output Type Description
delete EventEmitter Event emitted on delete button click.
