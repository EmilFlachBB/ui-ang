Component: TabGroup

TabGroupComponent represents component that is used as a wrapper for tab item components. It tracks user interaction and notifies parent upon selection change

Selector: bb-tab-group-ui

Inputs

Input Type Description Default
initialSelection number | undefined Setter (Input) for initial tab item selection -

Outputs

Output Type Description
select EventEmitter Event (Output) that emits new value once tab item selection has changed. As a parameter it provides index of newly selected item
