Component: ModeHeader

Component that displays a mode header.

Selector: bb-mode-header-ui

Inputs

Input Type Description Default
closeButtonAriaLabel string The aria label to describe the close button to screen reader users. -
reverseTitleOrder boolean Whether or not the order of title elements is reversed. true
secondaryTitle string The smaller, secondary title to display. -
showCloseButton boolean Whether or not to show the close button. Defaults to true. true
title string The main title to display. -

Outputs

Output Type Description
close EventEmitter Event fired when the close button is clicked.
