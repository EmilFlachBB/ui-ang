Component: CollapsibleAccordion

Component that groups Collapsible components to show them in an accordion fashion. It can assure that only one Collapsible panel can be opened at a time.

Selector: bb-collapsible-accordion-ui

Inputs

Input Type Description Default
closeOthers boolean Assures that only one Collapsible panel can be opened at a time. false
obeyParentAccordion boolean When true the Accordion will inherit the parent Accordion behaviour. -
openFirst boolean When true displays the first panel on load. false
