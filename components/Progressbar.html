Component: Progressbar

Component that represents the progress of a task.

Selector: bb-progressbar-ui

Inputs

Input Type Description Default
animated boolean A flag indicating if the stripes of the progress bar should be animated. Defaults to false. false
ariaLabel string Accessible label for the progressbar. 'progress bar'
height string | undefined Height of the progress bar. Can take values in px, em, rem. -
max number Maximum value of the progressbar. Defaults to 100. 100
showValue boolean A flag indicating if the current value should be displayed within the progressbar. Defaults to false. false
striped boolean A flag indicating if the progress bar should be striped. Defaults to false. false
type ProgressbarType | undefined Type of progress bar. Takes the values primary, success, info, warning & danger. -
value number Current value of the progressbar. Should always be less or equal to max. Defaults to 0. 0
