Component: LoadingIndicator

Component that displays a loading spinner and message.

Selector: bb-loading-indicator-ui

Inputs

Input Type Description Default
hasBackground boolean Defines whether gray background should be displayed. Default to false (background hidden). false
inline boolean Define if spinner should be shown inline without any default padding and margin. Default to false (show default padding and margin). false
loaderSize LoadingIndicatorSize The size of the loading indicator. Defaults to 'md'. 'md'
showDelay number Delay in milliseconds before the loading indicator display. Default to 0. 0
text string The text to display under the loading indicator. -
