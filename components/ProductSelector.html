Component: ProductSelector

A dropdown component that displays items in a pre-defined format. Every item from the list in the custom templates need to match the following requirements: * It should have a `role="option"` attribute for detection by Screen Reader like option. * It should have a `#listItem` hash tag for keyBoard accessibility (navigate with keyUp/keyDown) * It should use a `div` HTML element instead of a button for its host HTML element * It should have a `tabindex="0"` Here is a example of custom tempates for the list items: ``` ```
