Component: PaymentCard

A component that displays card of any vendor that can be used as a form of payment.

Selector: bb-payment-card-ui

Inputs

Input Type Description Default
displayEmblem boolean Display Backbase logo on the card. true
displayOverlay boolean Display card overlay in the card. Defaults to false. false
imageSrc string Configuration to pass a link or (base64, svg) image to be used as the background image of the card. Defaults to undefined. -
overlayIcon "access-time" | "lock" | "not-interested" | "remove-circle-outline" | "warning" Overlay icon to be displayed on the card. Possible values: 'access-time', 'lock', 'not-interested', 'remove-circle-outline', 'warning'. Defaults to undefined. -
overlayText string Overlay status to be displayed on the card. Defaults to undefined. -
paymentCard PaymentCard Payment card data. -
paymentCardNumberFormat PaymentCardNumberFormat Configuration of how the card number should be formatted. Defaults to the following format: Example :{ length: 16, maskRange: [0, 12], segments: 4, } DefaultPaymentCardNumberFormat
size "sm" | "md" Defines the size of the card. Small cards show no content. Defaults to 'md'. PaymentCardSizes.MD
