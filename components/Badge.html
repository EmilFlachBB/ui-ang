Component: Badge

Component that displays a badge.

Selector: bb-badge-ui

Inputs

Input Type Description Default
abbr string | boolean The abbreviation to show when using circle styling. false
ariaLabel string | undefined Accessible label when control does not need to render label tag. -

circle boolean Whether circle styling should be applied. false

wrap boolean Whether text wrap styling should be applied. false
