Component: Heading

Component that displays a header with navigation ### Displaying navigation options based on entitlements Heading component could render the navigation items based on entitlements. To enable this feature PERMISSIONS_STRATEGY token from `@backbase/ui-ang/heading` should be privided. F.e. ``` import { PermissionsStrategy } from '@backbase/foundation-ang/web-sdk'; //... providers: [ { provide: PERMISSIONS_STRATEGY, useClass: PermissionsStrategy } ] //.... ```

Deprecated

Selector: bb-heading-ui
