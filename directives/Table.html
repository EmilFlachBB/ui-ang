Directive: Table

Directive that displays a table. The table directive doesn't provide any specific properties for accessibility. The `aria-sort` tag can be set on the sortable header cells with ascending and descending values.

Inputs

Input Type Description Default
bbTable Array The data source for the table. -

Outputs

Output Type Description
rowClick EventEmitter Emits an event when a row is clicked. An sample implementation could be: Example :onRowClick(value: unknown) { ..... } This will require `[bbRow]="rowData"` to table row and `[bbTableFocus]` to root table element.
selectRow EventEmitter Emits an event when a row is selected.
sort EventEmitter An event emitter for on sort actions. An sample implementation could be: Example :onSort({ column, direction }: SortEvent) { ..... }This will require you to add [bbSortable]="'columnName'" to column header.
